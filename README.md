# ドローン在庫管理API（DroneInventorySystem）

## 概要
本プロジェクトは、研修課題として作成された製造業向け在庫管理システム（REST API）です。
部品および製品の在庫管理を想定した業務システム設計を通じて、Spring BootによるAPI開発と設計思想の理解を目的としています。

---

## 目的
- REST API開発の基礎理解
- 業務システムを想定した設計理解
- 三層構造アーキテクチャの習得
- データ設計およびトランザクション管理の理解

---

## システム概要
製造業における部品・製品の在庫管理を想定し、以下の業務を扱う設計となっています。

- 部品・製品の在庫管理
- 入荷・出荷プロセス管理
- 在庫履歴の管理（トレーサビリティ）

---

## 使用技術
- Java
- Spring Boot
- MySQL

---

## 設計ドキュメント
本プロジェクトでは研修資料として以下の設計ドキュメントが提供されています。

- APIシステム概要
- API仕様（共通）
- データ要件（ER図・テーブル設計）
- 部品在庫管理API設計
- 製品在庫管理API設計

（docs/設計 配下に格納）

---

## 学習内容
- Spring BootによるREST API構築
- Controller / Service / Repository構造の理解
- DB設計（ER図・テーブル設計）の理解
- 業務フローを意識したAPI設計の理解

本プロジェクトは研修課題として作成されたものであり、
